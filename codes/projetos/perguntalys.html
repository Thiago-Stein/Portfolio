Jogar agora

Jogo de raciocínio lógico para o público infantojuvenil, com progresso salvo em cookies locais. Desenvolvido usando HTML5, CSS e JavaScript.


O que o proejeto oferece:

Este projeto foi pensado para treinar lógica, atenção e memória, com uma interface simples e botões fáceis de usar. Ele também salva o progresso do jogador localmente para manter o avanço entre sessões.

  • Experiência responsiva com HTML5 e CSS
  • Interação dinâmica usando JavaScript
  • Progresso salvo em cookies no navegador
  • Design leve e foco na jogabilidade
